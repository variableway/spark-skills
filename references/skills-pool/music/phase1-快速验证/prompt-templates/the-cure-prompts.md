## Prompt 关键词速查

| 维度 | 关键词 |
|------|--------|
| 吉他音色 | chorus-drenched, shimmering, watery, arpeggiated clean |
| 贝斯 | melodic bass, prominent bass lines, driving bass |
| 合成器 | warm synth pads, DX7 textures, lush pads |
| 人声 | breathy baritone, emotional, wavering, soaring choruses |
| 制作 | reverb-heavy, layered, atmospheric, 1989 production |
| 和弦 | minor key, i-VII-VI-V, I-vi-IV-V, suspended chords |
| 速度 | 100 BPM (深沉) - 130 BPM (明亮) |
| 结构 | sparse intro building to full arrangement, extended outros |

## 测试步骤

3. 从变体中选择最接近 The Cure 风格的
4. 在最佳变体基础上微调（改 1-2 个关键词）
5. 重复 3-5 次直到满意
