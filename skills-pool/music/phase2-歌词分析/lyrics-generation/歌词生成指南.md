# 歌词生成指南

## 前提条件

- Phase 1 已验证 AI 音乐工具可用
- 已收集歌词并完成风格卡分析
- 已理解每位艺术家的风格特征

## 歌词生成流程

### Step 1：确定输入参数

在生成歌词前，明确以下信息：

```yaml
目标艺术家风格: [The Cure / Suede / Neil Young / Bob Dylan]
主题: [你想表达什么？一句话描述]
情绪: [忧郁/怀旧/浪漫/愤怒/温暖/讽刺]
场景: [城市/自然/回忆/未来/抽象]
人称: [第一人称/第二人称/第三人称]
歌曲长度: [短 2 verse / 中 3-4 verse / 长 5+ verse]
```

### Step 2：使用 AI 生成歌词

将风格卡 + 输入参数组合成 Prompt：

```
你是一个歌词作者。请基于以下风格卡和参数生成歌词。

## 风格参考
[粘贴对应艺术家的风格卡内容]

## 输入参数
- 主题：[你的主题]
- 情绪：[你的情绪]
- 场景：[你的场景]
- 人称：[你的人称选择]
- 长度：[你的长度选择]

## 要求
1. 严格遵循风格卡中的结构约束
2. 使用风格卡中列出的核心意象
3. 遵循该艺术家的韵律模式
4. 不要直接抄袭任何已有歌词
5. 生成 2 个版本供选择
```

### Step 3：人工微调

AI 生成的歌词通常需要微调：

| 常见问题 | 修改方法 |
|---------|---------|
| 太直白/缺乏意象 | 替换为更抽象的表达 |
| 意象堆砌/不自然 | 删除多余的，保留 2-3 个核心意象 |
| 韵脚生硬 | 重写该行，先确定意思再调韵脚 |
| 不像目标风格 | 对照风格卡检查缺失的元素 |
| 结构不对 | 按风格卡的结构模板重新排列 |
| 缺乏个人表达 | 加入你自己的经历/情感细节 |

### Step 4：配合 Prompt 生成音乐

将最终歌词填入 Phase 1 的 Prompt 模板：

```
[Style of Music]
(使用对应艺术家的 Prompt 模板)

[Lyrics]
[Verse 1]
(粘贴微调后的歌词)

[Chorus]
(粘贴微调后的歌词)

...
```

### Step 5：评价和迭代

用 Phase 1 的评价表打分，特别关注：
- 歌词与音乐的契合度
- 歌词是否传达了你想要的情绪
- 整体是否让你想反复听

## 跨风格生成

如果你想融合多种风格：

```
请融合以下两种风格生成歌词：

风格 A：[艺术家 A 的风格卡]
风格 B：[艺术家 B 的风格卡]

融合比例：[70% A + 30% B]
- 音乐风格跟随 [A]
- 歌词风格跟随 [B]
- 意象从两者中各取一半

主题：[你的主题]
...
```

## 歌词质量自检清单

- [ ] 读出来是否自然（不像 AI 生成的）
- [ ] 是否有至少 1 个让你印象深刻的句子
- [ ] 是否传达了你想表达的情绪
- [ ] 是否符合目标风格的核心特征
- [ ] 配上音乐后是否和谐
- [ ] 是否有个人化的元素（不是纯模仿）
