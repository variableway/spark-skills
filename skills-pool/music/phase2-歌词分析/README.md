# Phase 2：歌词分析 + 生成

**目标**：分析歌词模式，生成风格化歌词
**时间**：2-3 周
**依赖**：Phase 1 完成

## 步骤

### Step 1：收集歌词（3-5 天）
- 按收集清单收集歌词
  - [The Cure 10首](./lyrics/the-cure/收集清单.md)
  - [Suede 5首](./lyrics/suede/收集清单.md)
  - [Neil Young 5首](./lyrics/neil-young/收集清单.md)
  - [Bob Dylan 5首](./lyrics/bob-dylan/收集清单.md)
- 每首歌保存为单独的 Markdown 文件
- 按模板格式添加风格标注

### Step 2：AI 分析歌词（2-3 天）
- 每位艺术家的歌词一起发给 Claude/GPT
- 使用风格卡中的分析 Prompt
- 将分析结果填入风格卡

### Step 3：创建风格卡（1-2 天）
- [The Cure 风格卡](./style-cards/the-cure-风格卡.md)
- [Suede 风格卡](./style-cards/suede-风格卡.md)
- [Neil Young 风格卡](./style-cards/neil-young-风格卡.md)
- [Bob Dylan 风格卡](./style-cards/bob-dylan-风格卡.md)

### Step 4：歌词生成测试（3-5 天）
- 参考 [歌词生成指南](./lyrics-generation/歌词生成指南.md)
- 每位艺术家生成 2-3 首歌词
- 将歌词 + Phase 1 Prompt 一起生成音乐
- 评价歌词与音乐的契合度

### Step 5：微调迭代（3-5 天）
- 根据评价结果调整风格卡
- 优化歌词生成 Prompt
- 生成最终版本

## 成功标准

- [ ] 4 份风格卡完整填写
- [ ] 每位艺术家至少生成 2 首可用的风格化歌词
- [ ] 至少 2 首歌词配合 AI 音乐生成的效果让你满意
- [ ] 歌词有个人化元素，不只是对原风格的模仿
