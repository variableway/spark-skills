# 个人化歌词创作

在风格模仿的基础上，加入你自己的故事和情感。

## 为什么需要个人化

- 纯模仿 AI 风格的歌词会缺乏灵魂
- 你自己的经历和情感是独一无二的素材
- 最好的音乐是"熟悉的风格 + 真实的情感"

## 个人化方法

### 方法 1：真实故事框架

```
请基于以下风格卡生成歌词，但使用以下真实故事作为素材：

风格：[艺术家风格卡]

真实故事（1-3 句话概述）：
"[你的一段经历，不需要完整，只需核心情绪和场景]"

要求：
1. 不要直接复述故事
2. 将故事转化为该风格的意象和表达
3. 保留真实情感，但用艺术的距离感包装
4. 每段 Verse 可以是故事的不同角度或时刻
```

**示例输入：**
```
真实故事："去年秋天在一个陌生的城市，晚上独自走在河边，突然想起小时候的事。"

风格：The Cure
→ AI 可能生成：Verse 中出现河流、秋天、孤独的城市灯光，
   Chorus 回到童年的某个具体画面，
   整体情绪是忧郁的怀念
```

### 方法 2：情绪关键词

```
风格：[艺术家风格卡]

我现在的心情/想表达的情绪（用 3-5 个关键词描述）：
[关键词1], [关键词2], [关键词3]

请基于这些情绪关键词生成歌词。
```

**示例：**
```
关键词：释然, 深夜, 远方, 不舍

风格：Neil Young
→ AI 可能生成：关于深夜开车离开某个地方的歌，
   有释然也有不舍，配简单的和弦和口琴
```

### 方法 3：场景 + 意象

```
风格：[艺术家风格卡]

我脑海中的画面：
[描述一个具体的画面，可以是真实的也可以是想象的]

请将这个画面转化为歌词。
```

**示例：**
```
画面："雨后的天台上，远处的霓虹灯模糊了，只有吉他声在回响"

风格：Suede
→ AI 可能生成：Urban decay 主题，
   雨水和霓虹的意象，戏剧化的情感
```

### 方法 4：对话式

```
风格：[艺术家风格卡]

请写一首歌，是 [人物A] 对 [人物B] 说的，
核心想表达的是：[一句话]
```

**示例：**
```
人物A：一个离开的人
人物B：留在原地的人
核心："我不确定离开是对是错"

风格：Bob Dylan
→ AI 可能生成：Verse-only 结构，
   每段是一个角度的反思，带反问和隐喻
```

## 个人化自检

- [ ] 歌词中有至少 1 个只有你才会写的细节
- [ ] 听这首歌时，你能联想到自己的真实经历
- [ ] 别人听这首歌时，能感受到真实的情感（不只是技术模仿）
- [ ] 你愿意把这首歌分享给朋友（不觉得尴尬）
