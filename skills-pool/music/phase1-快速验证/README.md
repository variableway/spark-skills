# Phase 1：快速验证

**目标**：验证 "用 AI 生成你喜欢的音乐" 是否可行
**时间**：1-2 周
**工具**：Suno（主）+ Udio（辅）

## 步骤

### Step 1：选择工具并注册
- [ ] 注册 Suno 账号（suno.com）
- [ ] 注册 Udio 账号（udio.com）
- [ ] 熟悉两个工具的基本操作

### Step 2：选择参考歌曲
每位艺术家选 1 首参考歌曲：

| 艺术家 | 参考歌曲 | 风格变体 |
|--------|---------|---------|
| The Cure | Lovesong | 抒情 Gothic Pop |
| Suede | Animal Nitrate | 爆发力 Britpop |
| Neil Young | Heart of Gold | 经典民谣摇滚 |
| Bob Dylan | Blowin' in the Wind | 纯民谣 |

### Step 3：使用 Prompt 模板生成
- 进入 [prompt-templates/](./prompt-templates/) 选择对应艺术家的模板
- 每位艺术家先用基础 Prompt 生成一版
- 再尝试 2-3 个变体
- 记录到 [评价表](./evaluation/评价表.md)

### Step 4：迭代优化
- 从每组的变体中选择得分最高的
- 在最佳变体基础上微调 1-2 个关键词
- 每位艺术家迭代 3-5 次

### Step 5：总结
- 哪位艺术家的风格最容易用 AI 还原？
- 哪个工具（Suno vs Udio）效果更好？
- 最佳 Prompt 有什么共同特征？

## 成功标准

> 至少有 1 首生成的音乐让你觉得"有点像那个艺术家的味道"，且你愿意反复听。
