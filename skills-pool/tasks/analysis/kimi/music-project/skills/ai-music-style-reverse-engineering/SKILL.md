# AI 音乐风格逆向工程

## 适用场景

- 你喜欢某个艺术家/乐队，想生成类似风格的 AI 音乐
- 你想创建个人化的"风格配方库"
- 你想让 AI 生成的音乐更符合特定审美偏好

## 核心原则

1. **输入质量 > 数量**: 5-8 首精心分析的代表作 > 50 首泛泛而听
2. **结构化数据才能被 AI 利用**: 原始音频不如歌词+和弦+结构化特征
3. **迭代验证**: 生成→评估→调整→再生成，不要指望一次成功
4. **合法性**: 只使用公开可获取的歌词和和弦谱，不下载盗版音频

## 完整流程

### Phase 0: 选定目标

```
□ 选定 1 个首要目标艺术家/风格
□ 选定 AI 生成工具（Suno/Udio/其他）
□ 了解该工具的基本 Prompt 语法
```

### Phase 1: 素材收集

**收集清单（按优先级）**:

| 素材类型 | 获取来源 | 重要性 | 合法性 |
|----------|----------|--------|--------|
| 歌词文本 | Genius, AZLyrics, 官方歌词本 | ⭐⭐⭐ 核心 | ✅ 高 |
| 和弦谱 | Ultimate Guitar, Songsterr | ⭐⭐⭐ 核心 | ⚠️ 中 |
| 歌曲列表/排名 | 音乐杂志、乐评网站 | ⭐⭐ 辅助 | ✅ 高 |
| 音频特征 | Spotify API（需开发账号） | ⭐⭐ 辅助 | ✅ 高 |
| 背景故事 | 采访、纪录片、乐评 | ⭐⭐ 辅助 | ✅ 高 |
| MIDI/音频文件 | 自行扒谱或购买 | ⭐ 可选 | ⚠️ 注意版权 |

**保存格式**:
```
artist-name/
├── song-list.md          # 歌曲清单+时代划分
├── lyrics.md             # 歌词合集
├── chords.md             # 和弦/曲式分析
├── audio-features.md     # 结构化音频数据
└── raw-notes/            # 原始笔记
```

### Phase 2: 风格分析

**分析维度**:

```
1. 音乐特征
   ├── 调性偏好（哪个调用得最多？大调还是小调？）
   ├── BPM 范围（快/中/慢分别是多少？）
   ├── 和弦模式（常用进行是什么？）
   ├── 曲式结构（有无副歌？Bridge 怎么用？）
   └── 音色签名（吉他效果器？贝斯角色？鼓风格？）

2. 歌词特征
   ├── 主题光谱（最常见的 3-5 个主题）
   ├── 高频词汇（出现最多的词有哪些？）
   ├── 修辞手法（隐喻？重复？矛盾？）
   ├── 句式特征（平均长度？简单/复杂？）
   └── 叙事视角（第一人称？对话？抽象？）

3. 情绪曲线
   ├── 整体情绪倾向
   ├── 情绪发展方式（build up? up-down-up?）
   └── 核心矛盾（如：明亮旋律+阴郁歌词）
```

**输出物**: `style-recipe.md`（风格配方文档）

### Phase 3: Prompt 生成与迭代

**Prompt 三层结构**:

```
Level 1: 风格标签（工具 Style 栏）
└── 例: "The Cure, post-punk, 1980s, chorused guitar, melodic bass"

Level 2: 音乐描述（工具 Description 栏）
└── 例: "A minor, 140 BPM, verse-chorus structure, bittersweet atmosphere"

Level 3: 歌词指令（Custom Lyrics 模式）
└── 用 LLM 生成符合风格的歌词，包含指定高频词和修辞
```

**迭代检查表**:

| 检查项 | 评估 |
|--------|------|
| 调性是否匹配？ | |
| BPM 是否匹配？ | |
| 乐器音色是否匹配？ | |
| 歌词风格是否匹配？ | |
| 整体情绪是否匹配？ | |
| 与参考歌曲的相似度（1-10）| |

**常见问题与对策**:

| 问题 | 对策 |
|------|------|
| 太 generic | 添加更具体的年代/地域/效果器描述 |
| 太现代 | 禁用现代制作词汇，强调年代特征 |
| 歌词不对 | 在歌词 Prompt 中指定高频词和禁用词 |
| 乐器不对 | 明确描述每种乐器的音色和角色 |

## 快速启动模板

### 最小可行流程（30 分钟）

```
1. 选定艺术家 + 3 首最熟悉的歌（5 分钟）
2. 搜索歌词+和弦，复制到文档（10 分钟）
3. 基于熟悉度写出"我认为这首歌的特征是什么"（10 分钟）
4. 用以下内容生成第一个 Prompt（5 分钟）:
   "[艺术家名] style, [调性], [BPM], [乐器描述], [情绪], [年代]"
```
